#### 1st step 
```
git clone https://github.com/ratnesh003/HCLTech-Tasks.git
```

 #### 2nd step
 ```
 cd "HCLTech-Tasks/Silver Badge Assignments/Assignment 3/backend"
 ```

 ### 3rd step
 ```
 py -m venv venv
 ```

 ### 4th step
 ```
 venv\Scripts\activate
 ```

 ### 5th step
 ```
 pip install -r requirements.txt
 ```

 ### 6th step
 ```
py app.py
```
